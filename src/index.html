Projects

VR for daylighting analysis in architectural scenes

The focus of this project was to investigate how effective HDRP is in rendering lighting in a VR architectural environment. Research was done to gain insight into various techniques that had already been tested within this field. This research was then used to select the tools to be used to create the VR app. Interactions were programmed and lights were baked within Unity in an attempt to replicate realism in the scene. The main challenges faced was how graphically intensive rendering these scenes are and how compromises had to be made for the app to be run on specific hardware used and where it was tested on. Therefore, a limitation of this project was a trade-off between performance and graphical quality. A number of these issues were solved, as there were workaround methods available (see implementation). The implementation of the VR app provides an immersive experience and the techniques utilised in this project can be replicated in future design projects to achieve an environment that is realistic and modifiable.

Skills: VR, Unity3D, C#

2D Shooter (Made in Unity)

The type of game created is a third person action shooter. The core mechanic of the game is to use weapons(guns) to inflict damage upon enemies, thus defeating them when their HP hits zero to open the closed door(s) in their room. The map is a maze of sorts; there are multiple rooms in which several enemies reside. When the enemies are defeated, the door(s) behind them opens which allows further exploration of the map. There is no indication of where the player should go so there is a sense of mystery and excitement for the player. Upon reaching the final room, there is a certain key that the player can press to complete the level.
